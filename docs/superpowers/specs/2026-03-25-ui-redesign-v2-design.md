# UI Redesign V2: Agent App Container

## Overview

基于最新线稿重新设计 Alva Agent 桌面应用，从 chat-centric 转向 task-centric 体验。
目标：打造一个专业的 Agent App Container，支持任务分配、协作、管理。

### Welcome 区域设计要素

- **Logo 区域**: 居中，上方留白，品牌标识占位
- **标题**: "开始协作"，大字号，居中
- **副标题**: "7x24 小时帮你干活的全场景个人助理 Agent"，灰色小字
- **输入框**: 居中，宽度约 60% 主区域
  - 内含 placeholder: "分配一个任务或提问任何问题"
  - 底部工具栏: Model 选择器(左) | 附件按钮 + 技能按钮(中) | 发送按钮(右)
- **快捷操作卡片**: 4 个卡片水平排列
  - 制作幻灯片、数据分析、教育学习、创建网站
  - 点击后自动填入预设 prompt 并发送

## Component Architecture

### 新增组件

| 组件 | 类型 | 说明 |
|------|------|------|
| `WelcomeView` | View | 欢迎/首页状态，含 logo、标题、输入框、快捷卡片 |
| `QuickActionCard` | Helper | 快捷操作卡片（制作幻灯片等） |
| `NavItem` | Helper | 侧边栏导航项（图标+文字+hover） |
| `TaskHistoryItem` | Helper | 任务历史条目（名称+时长+状态） |
| `SettingsNav` | Helper | 设置对话框左侧导航 |

### 修改组件

| 组件 | 变更 |
|------|------|
| `Sidebar` | 完全重写，新导航结构 |
| `ChatPanel` | 条件渲染：无选中 → WelcomeView，有选中 → Chat |
| `InputBox` | 新增 Model 选择器、技能按钮 |
| `SettingsPanel` | 左导航 + 右内容的分类设置 |
| `SettingsDialog` | 更大尺寸适配新布局 |

### Style 复用策略

1. **Theme tokens**: 所有颜色通过 `Theme` 统一管理
2. **Helper functions**: 复用的 UI 片段抽为 `fn render_xxx(theme, ...) -> impl IntoElement`
3. **Consistent spacing**: px/padding 使用统一尺寸（px(4), px(8), px(12), px(16)...）
4. **Component patterns**: 按钮/卡片/导航项通过 helper 函数统一样式
