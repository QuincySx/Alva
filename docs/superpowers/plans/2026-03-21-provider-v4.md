# Provider V4 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the entire `ports/llm_provider` + `adapters/llm/openai_compat` with AI SDK V4-aligned provider interfaces and direct HTTP implementations for Anthropic + OpenAI.

**Architecture:** Bottom-up: types first, then traits, then HTTP layer, then providers, then migrate consumers, then delete old code.

**Tech Stack:** Rust, reqwest, serde, tokio, futures, async-trait, chrono, uuid

**Spec:** `docs/superpowers/specs/2026-03-21-provider-v4-design.md`
